## Install

just run `yarn`

## Usage

yarn start [options]

```
Options:
  Options:
  -c, --check        check time slots
  -l, --list         full timeslots output
  -p, --polling      long polling time slots
  -a, --addr <type>  search your address
  -h, --help         display help for command
```
